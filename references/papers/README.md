## Usage Guide

### Accessing Papers
All papers are organized into two main directories:
- `neuromorphic/` - Papers on neuromorphic computing
- `infinite_context/` - Papers on infinite context AI

### Searching Papers
Use the paper tracker (`paper_tracker.md`) to find papers by:
- Topic/subcategory
- Year
- arXiv ID
- Relevance score

### Downloading Papers
To download all papers:
```bash
cd references/papers
python3 download_papers.py
```

### Metadata
Paper metadata is available in:
- `paper_collection.json` - Structured JSON format
- `paper_tracker.md` - Markdown table format

## Citation Information

When citing papers from this collection, please use the following format:

**For arXiv papers:**
```
Author(s). (Year). Title. arXiv preprint arXiv:ID.
```

**For published papers:**
```
Author(s). (Year). Title. Journal/Conference Name, Volume(Issue), Pages.
```

## Key Research Areas Covered

### Neuromorphic Computing
1. **Core Concepts**: Brain-inspired computing, neuromorphic architectures
2. **Spiking Neural Networks**: Training, architectures, applications
3. **Hardware**: Memristors, RRAM, analog computing, photonics
4. **Learning**: Biologically plausible algorithms, STDP, Hebbian learning
5. **Applications**: Vision, robotics, edge computing
6. **Efficiency**: Energy-efficient computing, low-power AI

### Infinite Context AI
1. **Long Context**: Infinite attention, extended context windows
2. **Memory Systems**: Memory-augmented networks, episodic memory
3. **State Space Models**: Mamba, structured SSMs
4. **Retrieval**: RAG systems, retrieval-augmented generation
5. **Compression**: Context compression, efficient encoding
6. **Architectures**: Recurrent memory, attention alternatives

## Research Trends & Insights

### Neuromorphic Computing Trends
- **Hardware-Software Co-design**: Increasing focus on co-optimizing algorithms and hardware
- **Energy Efficiency**: Major emphasis on reducing power consumption
- **Biological Plausibility**: Growing interest in brain-inspired learning rules
- **Photonic Computing**: Emerging as a promising alternative to electronic neuromorphic systems
- **Edge Deployment**: Focus on deploying neuromorphic systems on edge devices

### Infinite Context Trends
- **Beyond Transformers**: Exploration of alternatives to standard attention mechanisms
- **Hybrid Approaches**: Combining retrieval, compression, and memory systems
- **State Space Models**: Rapid development of Mamba and related architectures
- **Test-Time Compute**: Scaling inference through reasoning and search
- **Multimodal Integration**: Extending long context to vision-language models
